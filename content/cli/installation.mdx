How to install the `depot` CLI on all platforms, with links to CI configuration guides.

## Mac

For Mac, you can install the CLI with Homebrew:

```shell
brew install depot/tap/depot
```

Or download the latest version from [GitHub releases](https://github.com/depot/cli/releases).

## Linux

Either install with [our installation script](https://depot.dev/install-cli.sh):

```shell
# Install the latest version
curl -L https://depot.dev/install-cli.sh | sh

# Install a specific version
curl -L https://depot.dev/install-cli.sh | sh -s x.y.z
```

Or download the latest version from [GitHub releases](https://github.com/depot/cli/releases).
